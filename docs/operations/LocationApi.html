This is the API

getLocation

Attain the locations of the data referenced with the data-ids, at which the data may be retrieved.

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/data/{data-ids}/locations

HTTP Method

GET

Response Type

Parameters

  • Path: data-ids Identifiers

    An array of data identifiers to operate upon. All data must be available in order to carry out a successful operation, otherwise no operation is carried out.

getNodeDataIdentifiers

Get a full listing of all data identifiers existing at a particular node

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes/{node-id}/data

HTTP Method

GET

Response Type

Identifiers

Parameters

  • Path: node-id Identifier

    The identifier of the relevant node

getNodeLocation

Get location of the node specified with node-id

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes/{node-id}/location

HTTP Method

GET

Response Type

Location

Parameters

  • Path: node-id Identifier

    The identifier of the relevant node

getNodes

Get all of the node locations and their identifiers known to the location service, either for replication of the location service, or for scheduling new computations.

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes

HTTP Method

GET

Response Type

Nodes

Parameters

rmNode

Delete information associated with the node-id from the location service, killing the node if it is still live.

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes/{node-id}

HTTP Method

DELETE

Response Type

Parameters

  • Path: node-id Identifier

    The identifier of the relevant node

signalData

Informs the location service that there is data available or being processed at the location of node-id.

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes/{node-id}/{data-ids}

HTTP Method

PUT

Response Type

Parameters

  • Path: node-id Identifier

    The identifier of the relevant node

  • Path: data-ids Identifiers

    An array of data identifiers to operate upon. All data must be available in order to carry out a successful operation, otherwise no operation is carried out.

signalLocation

Informs the location service that a node is available for work, and seeks a new node identifier.

URL

https://virtserver.swaggerhub.com/j-c/largerscale/1.0.0/nodes

HTTP Method

POST

Response Type

Identifier

Parameters

  • Body: body String(string)

    The location of the node that is available for work and seeking association to a node identifier
